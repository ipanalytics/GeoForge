Open-source GeoIP build pipeline

GeoForge

Consensus GeoIP compiler for reproducible local MMDB builds.

View on GitHub Output schema

Multi-source consensus

Merge DB-IP, GeoLite2, IP2Location, Sypex, operator geofeeds, RIR delegated stats, and GeoNames reference data into a local MMDB.

Local and reproducible

Source downloads are checksum-tracked. Builds are skipped when inputs have not changed, and every release receives a quality report.

Operational output

Records include matched prefix, country metadata, registry country, city, postal code, coordinates, time zone, confidence, and accuracy radius.
